# GalleryUI-WPF

基于 **WPF-UI** 开发的 **gallery-dl** 可视化图形界面工具，无需记住命令行，轻松下载漫画与图片。

## ✨ 功能特色

- 🖥️ **直观的图形界面**：告别命令行，通过输入框和按钮即可完成下载
- 🎨 **现代化设计**：基于 WPF-UI，提供 Windows 11 风格的原生界面

前往 [Releases](https://github.com/chifanoh/GalleryUI-WPF/releases) 页面下载最新版本。

## 🛠️ 技术栈

| 技术 | 用途 |
| :--- | :--- |
| .NET 10 + WPF | 桌面应用框架 |
| [WPF-UI](https://github.com/lepoco/wpfui) | 现代化 UI 组件库 |
| [gallery-dl](https://github.com/mikf/gallery-dl) | 核心下载引擎 |

## 📋 依赖说明

本工具依赖 `gallery-dl.exe` 来执行下载任务。软件包内已包含该程序，无需额外安装。
